# Wafer-Yield-Calculator
This is a calculator that uses the Poisson yield equation to calculate wafer yield. Inputs are defect density and die area.

This is my first official Python project that is not class work. I am trying to improve my coding skills, as well as machine learning and AI. This account will be a reflection of that progress.

I decided to make this calculator because in the field of semiconductor manufacturing, many companies are driven by the desire to increase wafer yield. Increasing wafer yield makes for a more efficient use of resources, and is a good indicator of how well the manufacturing process is running. The closer to 100% as possible, the better.
